## 프로젝트 실행

1. 루트 디렉토리에 .env 파일을 생성하여 API 주소를 저장

```
REACT_APP_URL={API 주소}
```

2. 의존성 패키지 설치

```
$ npm i
```

3. 프로젝트 실행

```
$ npm start
```
